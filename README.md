# udemy-cpp20-masterclass-project-template

* Visual Studio Code

### Compiler (Windows):
* GCC/G++
* Clang/LLVM
* MSVC

### Compiler (Linux):
* GCC/G++
* Clang/LLVM

### Debugger:
* GDB
* LLDB
